Ben je 2e jaar student? Doe de Minor

Minor Webdevelopment is voor iedereen te volgen van CMD, maar met sommige keuzeprojecten wordt het wel makkelijker om de minor succesvol af te ronden.

Welk project volg je / ga je volgen?

Project Tech Project Visual Project Concepting Project Interaction

Project Tech

Wanneer je project Tech hebt gekozen als keuzeproject in het 3e blok van jaar 2 heb je de beste aansluiting voor de minor. De vakken die worden gegeven bij de minor gaan verder waar project Tech ophield. Studenten die project Tech hebben gekozen hebben over het algemeen minder moeite met de minor dan studenten van andere keuzeprojecten.

Project Visual

Wanneer je project Visual gekozen hebt als keuzeproject in het 3e blok van jaar 2 heb je achterstand op een student die project Tech heeft gekozen, maar dat betekend niet dat het onmogelijk is om de minor niet te halen. Wanneer je aan kan tonen dat je over voldoende kennis en motivatie beschikt om de minor te doen is de minor prima te halen.

Wanneer je project Visual hebt gekozen betekend het niet dat je op alles achter loopt op andere studenten. In vakken zoals CSS To The Rescue en Web Design komen namelijk ontwerpprincipes terug die jij als Visual student waarschijnlijk beter kent en weet dan een student die geen project Visual hebt gekozen.

Het kan handig zijn om voor jezelf de basis van Node.JS te leren, aangezien deze programmeertaal veel terug zal komen.

Wanneer je aan kan tonen dat je over voldoende kennis en motivatie beschikt om de minor te doen is de minor prima te halen.

In de korte projecten en tijdens de vakken moet je namelijk ook zelf concepten bedenken om uit te werken, iets wat jij als concepting student waarschijnlijk beter weet dan een student die geen concepting heeft gedaan.

Het kan handig zijn om voor jezelf de basis van Node.JS te leren, aangezien deze programmeertaal veel terug zal komen.

Project Interaction

Wanneer je project Interaction gekozen hebt als keuzeproject in het 3e blok van jaar 2 heb je achterstand op een student die project Tech heeft gekozen, maar dat betekend niet dat het onmogelijk is om de minor niet te halen.
